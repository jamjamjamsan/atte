# Atte

Atteは勤怠管理システムです。従業員の出退勤および休憩の打刻を行えます。


## デモ

会員登録またはログイン後、出退勤および休憩の打刻ページに
遷移。日付一覧では利用者の出勤表の一覧を5件ずつ表示させます。
